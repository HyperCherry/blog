    校园学习平台策划 #

    背景与目的 #

    当代的诸多大学生尤其是大学应届学生，在学习、证件考取、学位资历等方面，存在认知缺乏。同时缺乏一种行之有效的一种学习方式，获得的指导并匮乏专业性，缺乏彼此与老师的沟通与督促。

    运行模式 #

    通过小程序、网站、APP等多种方式建立一个具有泛用性的校内学习社区，解决与专业老师沟通以及彼此之间的沟通难点与痛点

    具体细节 #

    先由一个学生创建并发起「学习圈」，当「学习圈」所指向的「项目」数量>1时，「项目」将会被自动创建。后续学生在自愿的前提下加入此「学习圈」
    由建立者向意向老师发出邀请，在老师同意邀请之后，老师通过对应的程序完成对「学习圈」的绑定。老师能够选择性的对「学习圈」收取绑定费用。
    由「建立者」与「圈内协管」协商过后，可以决定此「学习圈」的性质，确定是否需要老师协助。
    在达到目标期限时，此「学习圈」将会解散并计入学习记录。（长期「学习圈」除外），以此达成资源的循环
